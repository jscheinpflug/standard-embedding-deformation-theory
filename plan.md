# Deformation theory of the standard embedding

## Background
- In Type II string theory on Calabi-Yau manifold, we believe that there exists a moduli space of vacua parametrized by non-obstructed exactly marginal deformations (a conformal manifold of CFTs) that correspond to 2d (2,2) SCFTs
- In heterotic string, the standard embedding vacua are also 2d (2,2) SCFTs, but in contrast to the TypeII case, a generic heterotic vacuum is not of this type, but rather is a chiral 2d (2,0) SCFT, and it seems that generically there is no conformal manifold
- There is a large body of work on heterotic deformations in the geometric regime of NLSM + worldsheet instanton corrections. We are not interested in that here, we are interested in deformations of intrinsically defined CFTs.
- The question is what is the space of these chiral (2,0) SCFTs. A suitable starting point for exploration seem to be the standard embedding (2,2) vacua and deforming away from them. The key question is which deformations are obstructed and which are not.
